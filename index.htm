Asteroids: The main feature of the classic Asteroids Game is its game physics. First, the rotation of the ship when pressing the left or right key. It is done by using the rotation matrix. Second, when press the thrust key, the ship needs to accelerates, done by multiplying the percentage of the acceleration value to the current velocity. Thirdly, it needs to glide for a period of time after player release the thrust key. It is done by multiplying the deceleration percentage to the current velocity. When an asteroid hits by a bullet, it will become a smaller asteroid, and if hit the second time, it will be destroyed. Ship will have 3 lives, each time it gets destroyed, it will spawn at the center with a short invincible period.

BubbleMan: Similar to BomberMan, the main mechanic of the game is to drop bubbles to attack. Agents can drop bubbles on the ground, and it will explode after a peroid of time. If an agent is blasted when bubble exploded, that agent will be killed. If there is a chest blasted then that chest will blow up and have a chance to spawn an item. There are 2 types of items, one for the agent to be able to drop more bubbles, and the other one for the agent's bubbles to blast longer. There are 3 AIs controlled by computer to be your enemies. They will attack you if there is an open path to you, if not they will blow up the chests and get items. The AIs also able to avoid being blasted b calculating the save areas. I programmed the AIs using A* pathfinding algorithms. The player will win if all enemies are dead.
